
# e-utils

# 初衷 

因为，在 play code 的过程中，总是要用到以前写的东西，比如七牛的封装啊，wait 函数啊这些，又懒得复制。 

所以，决定做成一个 npm 包，发在上面然后要用的时候 require 一下就可以了。 

# 使用方法 

安装 

``` bash
$ npm i --save eczn-utils 
```

引用 

``` js
const E = require('eczn-utils'); 
let { wait, qnx } = E; 
```
